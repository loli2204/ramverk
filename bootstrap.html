Om Bootstrap

Bootstrap är ett populärt front-end ramverk som används för att skapa responsiva och snygga webbsidor och webbapplikationer.

För mer information om Bootstrap, besök Bootstrap webbplats.

Jag tyckte att bootstrap var mycket enkelt att installera då jag använde tillägget i vscode. Det var lätt att snabbt styla sidorna så de såg bra och proffessionell ut. Jag tycker det är mycket bra att det är ett responsivt system som anpassar sidan efter olika skärmstorlekar. Det finns möjlighet att ändra och anpassa utseendet efter önskemål. Det finns många färdiga mallar att använda sig av. Nackdelarna med bootstrap att det lätt kan skapas många sidor som är väldigt lika varandra.
