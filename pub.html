Publicering

Webbplatsen är publicerad med GitHub Pages. På GitHub Pages går det att publicera statiska webbplatser med HTML, CSS och JS samt bilder. Repon får däremot inte vara över 1GB och det är inte tillåtet att använda GitHub Pages i kommersiellt syfte. Det är mycket lätt att publicera sidan med GitHub Pages. Först skapar man ett nyy repo och laddar upp filerna. Därefter går man till settings och klickar sig vidare till Pages. Där väljer man den branch som ska publiceras och klickar på spara. Därefter läggs sidan upp live efter ett par minuter. Källa.
